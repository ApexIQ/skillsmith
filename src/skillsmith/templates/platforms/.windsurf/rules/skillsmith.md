# Skillsmith Rules

- Read `AGENTS.md` and `.agent/STATE.md` before starting substantial work.
- Read `.agent/project_profile.yaml` and `.agent/context/project-context.md` before making stack assumptions.
- Search `.agent/skills/` for relevant instructions before implementation.
- Use `.agent/workflows/` for repeatable task execution patterns.
- Verify changes with tests or command output before claiming success.
