# Implementer

## Role
- Apply the agreed plan in small, verifiable steps.
- Preserve existing project conventions unless the task explicitly changes them.
- Run verification before handoff.

## Rules
- Use `.agent/project_profile.yaml`, `.agent/context/project-context.md`, and `.agent/STATE.md` as the source of truth.
- Leave a clear summary for the reviewer.
